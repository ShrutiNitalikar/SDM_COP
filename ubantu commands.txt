pwd - showcase current location, talks about path working directory
whoami - shows the name of user
mkdir directory_name - creates a directory
ls - it shows a file in a folder (if created)
rmdir directory_name - removes the directory 
ls -lt : gives name of file, directory with details (latest is first)
ls -ltr : gives name of file, directory with details in reverse order (latest is last)
date - gives todays date  Thu Sep  4 11:52:21 PDT 2025
date +%D - 09/04/25
date +%T - showcase time (11:53:39)
date +%H:%M -displays time (only in hrs and minutes)
touch ABC.txt - creates file named abc.txt
nano ABC.txt - opens new window that is editor, write 2 3 lines in the editor, ctrl+x quits the editor and opens ubantu again
cd .. -
cd -
ls
clear - clears everything
